# FPS 射击游戏 — Windows 10 部署指南

> 这份指南会一步一步教你怎么在自己的电脑上运行这个游戏。
> 就像搭积木一样，跟着步骤来就行！

---

## 你需要准备什么

- 一台 Windows 10 电脑
- 能上网（用来下载软件）
- 一个浏览器（推荐 Chrome 或 Edge）

---

## 第一步：安装 Node.js

Node.js 是让游戏服务器运行起来的工具，就像游戏的"发动机"。

1. 打开浏览器，访问：https://nodejs.org/zh-cn
2. 你会看到页面上有一个绿色大按钮，写着 **"LTS"（长期支持版）** — 点击下载
3. 下载完成后，双击打开安装包（文件名类似 `node-v20.xx.x-x64.msi`）
4. 安装过程：
   - 一路点 **"Next"（下一步）**
   - 遇到 "I accept" 的勾选框，**打勾**
   - 最后点 **"Install"（安装）**
   - 等一会儿，点 **"Finish"（完成）**

### 验证安装成功

1. 按键盘上的 `Win + R`（Win 就是键盘左下角那个 Windows 图标键）
2. 在弹出的小窗口里输入 `cmd`，按回车
3. 在黑色窗口里输入下面这行命令，然后按回车：

```
node --version
```

4. 如果看到类似 `v20.12.0` 这样的版本号，说明安装成功了！

> 如果显示"不是内部或外部命令"，试试重启电脑再来一次。

---

## 第二步：获取游戏文件

### 方法 A：从压缩包获取（最简单）

如果别人给了你一个 `fps-game.zip` 压缩包：

1. 在桌面或者你喜欢的地方创建一个文件夹，比如叫 `我的游戏`
2. 把压缩包解压到这个文件夹里
3. 解压后你应该能看到这样的文件结构：
   ```
   fps-game/
   ├── server.js
   ├── package.json
   └── public/
       ├── index.html
       ├── game.html
       └── src/
           └── ...
   ```

### 方法 B：从 Git 获取（如果你有仓库地址）

1. 先安装 Git：访问 https://git-scm.com/download/win ，下载并安装
2. 打开 cmd（方法同上），输入：

```
cd 桌面
git clone <仓库地址> fps-game
```

> 把 `<仓库地址>` 替换成实际的网址。

---

## 第三步：安装游戏依赖

游戏需要一些"零件"才能运行，我们用一条命令就能自动安装。

1. 打开 cmd（按 `Win + R`，输入 `cmd`，回车）
2. 进入游戏文件夹。假设你把游戏放在了桌面：

```
cd 桌面\fps-game
```

> **小提示**：如果你不确定路径，可以在文件管理器里打开 fps-game 文件夹，
> 然后在地址栏点一下，复制路径，再粘贴到 cmd 里。
> 比如：`cd C:\Users\小明\Desktop\fps-game`

3. 输入下面的命令并回车：

```
npm install
```

4. 等待几秒到几十秒，你会看到一些文字在滚动，最后出现类似这样的内容：
   ```
   added 64 packages in 5s
   ```
   这就表示安装成功了！

> 如果报错"npm 不是内部命令"，说明 Node.js 没装好，回到第一步重新安装。

---

## 第四步：启动游戏

在同一个 cmd 窗口里（确保你还在 fps-game 文件夹），输入：

```
npm start
```

你会看到：
```
FPS Game server: http://localhost:3000
```

**恭喜！游戏服务器启动成功了！**

> ⚠️ 注意：这个黑色窗口不能关闭！关了游戏就停了。
> 把它最小化到任务栏就行。

---

## 第五步：打开游戏

1. 打开 Chrome 或 Edge 浏览器
2. 在地址栏输入：

```
http://localhost:3000
```

3. 按回车，你就能看到游戏主菜单了！

---

## 第六步：开始玩

### 主菜单

你会看到两个关卡可以选择：

| 关卡 | 名称 | 内容 |
|------|------|------|
| 01 | 混合战场 | 地面战，消灭步兵和飞行敌人 |
| 02 | 海上决战 | 海战，操炮击沉敌舰 |

点击任意一个关卡卡片进入游戏。

### 操作说明

| 按键 | 功能 |
|------|------|
| W | 向前走 |
| S | 向后退 |
| A | 向左走 |
| D | 向右走 |
| 空格 | 跳跃 |
| Shift | 冲刺（按住跑得更快） |
| Ctrl | 蹲下 |
| 鼠标移动 | 瞄准/转身 |
| 鼠标左键 | 射击 |
| 鼠标右键 | 瞄准镜（更精确） |
| R | 换弹夹 |
| 1 / 2 / 3 | 切换武器 |
| F | 互动（第二关用来操炮） |
| ESC | 暂停游戏 |

### 第二关特别操作

在海战关卡中：

- 走到大炮旁边，屏幕下方会出现 **"按F操作 XXX炮"** 的提示
- 按 **F** 进入大炮瞄准视角
- 移动鼠标瞄准敌舰，左键开火
- 再按 **F** 退出大炮，继续走动

## 遇到问题怎么办

### 问题1：浏览器打开后是白屏

- 检查 cmd 窗口是不是还开着、显示着 `http://localhost:3000`
- 刷新浏览器（按 F5）
- 换用 Chrome 浏览器试试

### 问题2：npm install 报错

- 确认你在 fps-game 文件夹里（cmd 里能看到路径包含 fps-game）
- 确认文件夹里有 `package.json` 这个文件
- 试试关掉 cmd 重新打开再来

### 问题3：端口被占用

如果看到 `EADDRINUSE` 错误，说明 3000 端口被别的程序占了。

在 cmd 里输入：
```
set PORT=8080
npm start
```

然后在浏览器访问 `http://localhost:8080`

### 问题4：游戏卡顿

- 关掉其他程序（特别是其他浏览器标签页）
- 确保浏览器没有开太多标签页
- 游戏设置里（按 ESC 暂停）可以调低一些参数

### 问题5：没有声音

- 点击一下游戏画面（浏览器需要你互动后才能播放声音）
- 检查电脑音量是不是静音了
- 游戏暂停菜单里可以调音量

## 怎么关闭游戏

1. 关闭浏览器标签页
2. 回到那个黑色 cmd 窗口
3. 按 `Ctrl + C`（同时按住 Ctrl 键和 C 键）
4. 如果它问你 "终止批处理操作吗"，输入 `Y` 回车

## 下次想玩的时候

不需要重新安装了！只需要：

1. 打开 cmd
2. 进入游戏文件夹：`cd 桌面\fps-game`
3. 启动服务器：`npm start`
4. 浏览器打开：`http://localhost:3000`

就这么简单！

---

## 让同一个 WiFi 下的朋友一起看

如果你想让连着同一个 WiFi 的小伙伴也能打开你的游戏：

1. 按 `Win + R`，输入 `cmd`，回车
2. 输入 `ipconfig`，回车
3. 找到 **"IPv4 地址"**，类似 `192.168.1.xxx`
4. 告诉你的朋友在浏览器里输入：`http://192.168.1.xxx:3000`

> 注意：Windows 防火墙可能会弹出提示，点"允许访问"就行。

## 文件夹里都有什么（好奇宝宝看这里）

```
fps-game/
├── server.js          ← 游戏服务器（Node.js 程序）
├── package.json       ← 项目配置文件（记录需要什么零件）
├── public/            ← 游戏本体（网页+代码）
│   ├── index.html     ← 主菜单页面
│   ├── game.html      ← 游戏页面
│   ├── src/           ← 游戏源代码（JavaScript）
│   │   ├── main.js    ← 主程序
│   │   ├── player.js  ← 玩家控制
│   │   ├── enemy.js   ← 敌人 AI
│   │   ├── weapons.js ← 武器系统
│   │   ├── scene.js   ← 场景搭建
│   │   ├── turret.js  ← 炮位系统（第二关）
│   │   └── ...
│   ├── data/          ← 关卡配置数据
│   ├── style/         ← 界面样式
│   └── assets/        ← 模型、音效等资源
└── node_modules/      ← 自动安装的零件（不用管）
```
